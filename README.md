## ☁️ About The Project ☁️

To Do is a mobile application built using React Native.

Some key features include:

    ✅ Be able to display to-do items to user's profile.

<br />

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
